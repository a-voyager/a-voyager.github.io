Windows去掉桌面右击菜单多余项

发表于 2016-02-02 分类于 Windows

前言

在Windows桌面鼠标右击菜单中，通常会出现很多多余的选项，比如显卡相关设置，office相关功能，而这些选项大多数人是不会经常去使用它们的，放在这里反而碍眼，特别是对于强迫症来说这能忍么，而且这样会减缓系统加载速度，表现就是电脑卡的时候想右击刷新，可右击菜单会死活弹不出来…这篇文章就交大家如何去掉右击菜单中的多余选项.

打开注册表

首先需要打开系统注册表，使用Windows徽标+R键打开运行弹窗，输入 regedit打开注册表。

由此打开注册表:

展开注册表并删除

在左边的窗口依次展开 HKEY_CLASSES_ROOT\Directory\Background\shellex\ContextMenuHandlers

然后删除除开New所有即可，再次强调不要删 New

重启生效

重启或者注销电脑即可生效.
